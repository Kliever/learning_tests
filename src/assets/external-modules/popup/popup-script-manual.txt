Плагин поп-ап (модальное окно). Плагин позволяет создавать бесконечное количество модальных окон.

  HTML-структура

   <button type="button" class="js-open-popup-btn" data-popup-name="popup1">Попап1</button>

   <div class="popup js-popup popup1">
    <div class="popup__table" role="complementary">
      <div class="popup__table-cell js-popup-area">
        <div class="popup__window">
          <button type="button" class="popup__close js-popup-close" aria-lavel="Закрыть модальное окно">X</button>

          контент

        </div>
      </div>
    </div>
  </div>

  Класс - js-open-popup-btn - кнопка открытия попап. Этому классу добавляется дата-аттрибут data-popupname с названием попапа. Это название добавляется классом в саму оболочку попап (класс js-popup).

  Класс - js-popup - оболочка попапа, сюда классом добавляется название попапа из атрибута js-open-popup-btn открывающей его кнопки

  Класс - js-popup-area - затемненная зона, при клике на которую происходит закрытие попап

  Класс - js-popup-close - кнопка закрытия попапа. Если она не нужна, можно сделать ее прозрачной и размером в 1 пиксель.

        !!!  При открытии любого попап-окна оно получает класс _active, а также тег <body> получает класс _active-popup, который добавляет свойство overflow: hidden;

        !!! В скрипте указаны места, куда вставлять функции добавления и удаления скролла

  
  ДАТА-АТТРИБУТЫ

  data-popup-name - обязательный аттрибут добавляется к кнопке js-open-popup-btn, указывает название открываемого попапа (самому попапу js-popup добавляется класс с таким же именем, как и содержимое аттрибута).

  data-delay-open-close = необязательный аттрибут добавляется к каждому попапу (js-popup), может быть разным для каждого попап, устанавливается число, указывает на задержку срабатывания кода включения скролла в секундах. Синхронизируется со значением transition в css. Значение по умолчанию - 0.5, можно установить любое дробное; 

  data-open-time - аттрибут устанавливается тем попапам (js-popup), которые открываются самостоятельно по времени. Значение аттрибута - число секунд, спустя которое должен открыться попап после загрузки страницы. Аттрибут можно добавить и попапу, который открывается по кнопке. Так, например, если его не откроет пользователь, спустя указанное время попап откроется самостоятельно.
